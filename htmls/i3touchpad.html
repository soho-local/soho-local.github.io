Fix the issue of touchpad not working in i3wm

January 11, 2019

When first log into i3wm, I found that the touchpad did not work in click and scroll. I googled and get the command "synclient" to use for checking the running values of touchpad. Then type following commands to fix the issue.

synclient TapButton1=1
synclient TapButton2=1
synclient VertEdgeScroll=1
synclient HorizEdgeScroll=1

Or, just write them into i3 config file.
